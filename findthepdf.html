GitHub Token Setup

To enable PDF uploads, you need a GitHub Personal Access Token:

  1. Go to GitHub Settings → Personal Access Tokens
  2. Click "Generate new token (classic)"
  3. Give it a name like "FindPDF Upload"
  4. Select the "repo" scope (full control of repositories)
  5. Click "Generate token"
  6. Copy the token and paste it below

Security: Your token is stored only in your browser's local storage and never sent to any third party.
